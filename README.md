## CS50 Introduction to Databases with SQL

Exercises in this repo come from the [CS50 SQL Course 2024](https://cs50.harvard.edu/sql/)

### Course Structure
0. Querying
1. Relating
2. Designing
3. Writing
4. Viewing
5. Optimizing
6. Scaling

### Useful `sqlite3` Commands
```
# to display results in a columnar format.
.mode column

# to show the column names at the top
.headers on
```
